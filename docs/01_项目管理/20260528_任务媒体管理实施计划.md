> 创建日期: 2026-05-28
> 设计文档: [20260528_任务媒体管理页面设计.md](../02_技术文档/前端/20260528_任务媒体管理页面设计.md)
> 优先级: 高（核心用户体验）

---

## 一、目标

为 comic-share 新增「任务管理」页面，统一管理所有 pipeline 生成的媒体产物（图片/音频/视频），提供归档分组 + 树形导航 + 媒体画廊的浏览体验。

---

## 四、风险与依赖

| 风险 | 缓解 |
|------|------|
| 大文件（视频）静态服务性能 | 使用 FileResponse + Range 支持，后期可迁移到 CDN |
| 磁盘空间增长 | 归档删除时物理删除文件；后期可加清理策略 |
| Pipeline 中途失败产出不完整 | task status 标记为 failed，媒体画廊显示已有产出 |
