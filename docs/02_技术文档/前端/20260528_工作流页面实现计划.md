# 工作流页面实现计划

> 创建日期: 2026-05-28
> 状态: 计划阶段
> 路由: `/workflow`
> 参考: `Old/static-comic/src/services/runninghub.ts` + `Old/server/core/runninghub_client.py`

---

## 一、当前状态

`/workflow` 页面（WorkflowManager.vue）现为占位 UI：
- 从 `config.yaml` 注册表读取引擎 + 工作流列表
- 可切换默认引擎 / 单工作流引擎覆盖
- **无实际功能**：不能添加/删除/编辑工作流，不能触发任务

---

## 二、需求拆解

### 需求 A — 产出类型下拉框

在媒体画廊新增一个下拉框用于筛选产出类型（取决于使用的 JSON 工作流生成的产出类别）：

| 选项 | 对应工作流类别 | 产出说明 |
|------|---------------|----------|
| 三视图 | 角色三视图工作流（t2i + multi-angle） | 生成角色正面/侧面/背面图 |
| 分镜图 | 文生图工作流（t2i_default） | 生成单场景分镜图片 |
| 视频 | 图生视频工作流（i2v） | 生成动态视频片段 |

**实现要点**：
- 在 MediaGallery.vue 新增 `a-select`（`outputType`），与现有 `filter`（媒体格式）并列
- 后端 Task meta.json 记录 `workflow_category` 字段
- 媒体查询 API 支持 `?output_type=` 参数筛选

### 需求 B — 工作流页面去占位 + RunningHub 实际调用

#### B1. 后端：RunningHub 客户端迁移

从老项目 `Old/server/core/runninghub_client.py` 迁移核心能力：

| 函数 | 说明 | API |
|------|------|-----|
| `test_connection()` | 测试 API Key 连通 | `POST /task/openapi/create` (workflowId=0) |
| `fetch_workflow_json()` | 获取工作流节点 JSON | `POST /api/openapi/getJsonApiFormat` |
| `analyze_nodes_by_rules()` | 规则解析可调参数 | 本地计算，NODE_TYPE_MAP |
| `create_task()` | 提交任务 | `POST /task/openapi/create` |
| `poll_task_status()` | 查询状态 | `POST /task/openapi/outputs` |
| `cancel_task()` | 取消任务 | `POST /task/openapi/cancel` |
| `upload_image()` | 上传图片 | `POST /task/openapi/upload` |

**新建文件**: `backend/core/runninghub_client.py`

**config.yaml 新增配置**:
```yaml
runninghub:
  keys:
    consumer: ""
    enterprise_shared: ""
  active_key_type: consumer
  proxy: ""
  proxy_enabled: false
```

#### B4. 前端：WorkflowManager.vue 重构

**去除占位，实现完整交互**:

### RunningHub API 关键端点

| 功能 | URL | 备注 |
|------|-----|------|
| 获取工作流 JSON | `POST https://www.runninghub.cn/api/openapi/getJsonApiFormat` | apiKey + workflowId |
| 提交任务 | `POST https://www.runninghub.cn/task/openapi/create` | apiKey + workflowId + nodeInfoList |
| 查询状态 | `POST https://www.runninghub.cn/task/openapi/outputs` | code: 0=完成, 804=运行中, 813/812=排队 |
| 取消任务 | `POST https://www.runninghub.cn/task/openapi/cancel` | apiKey + taskId |
| 上传图片 | `POST https://www.runninghub.cn/task/openapi/upload` | multipart: apiKey + file |
